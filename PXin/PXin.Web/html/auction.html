相信APP用户竞拍服务协议

在“相信”APP平台依据《相信APP用户注册服务协议》注册的用户（即“拍卖人用户”和“竞买人用户”），在同意本协议以下全部条款后，方有资格享受“相信”APP平台提供的竞拍服务。您使用本平台提供的服务即意味着同意与本平台签订本协议并同意受本协议约束，使用竞拍服务前请认真阅读本协议。

特别声明：
特别声明：①“相信”APP平台在竞拍活动中的角色和作用仅限于为“拍卖人用户”和“竞买人用户”进行牵线和撮合，绝不直接参与前二者的实际拍卖和竞买；②“拍卖人用户”和“竞买人用户”在相信APP平台完成的竞拍交易所产生的包括但不限于所得税等均由“拍卖人用户”和“竞买人用户”自行自主申报，此等税赋与“相信”APP平台无关。

第一条：立约背景

为鼓励“相信”APP平台的用户作为拍卖人或竞买人积极参与竞拍活动，同时也为维护相信APP平台的竞拍活动秩序，规范拍卖人用户和竞拍人用户参与竞拍活动的行为，保障相关各方的合法权益，基于相应的竞拍流程与竞拍规则，和相信APP平台公示的规则，特制定本协议。

第二条：名词解释

2.1 “相信”APP平台：由广州相信科技有限公司研发、拥有所有权，且提供技术服务和运营管理。

2.2 拍卖人用户：在“相信”APP注册，按照“相信”APP平台相关规则中关于信息发布的要求发布竞拍品的用户。

2.3 竞买人用户：在“相信”APP注册，按照本协议及“相信”APP平台相关规则参与竞拍品竞价的用户。

2.4 竞拍品：拍卖人用户按照“相信”APP平台相关规则自行上传发布的、其本人用户账户体系下的个人数字资产，比如：V点、P点、SVC等“相信”APP内的数字资产。

第三条 参与竞拍的条件（成为拍卖人和竞买人的资格）

同时符合以下条件的用户（作为拍卖人和竞买人）方能参与平台提供的竞拍活动：

3.1 用户为“相信”APP平台会员，拥有独立的“相信”APP平台用户名。

3.2 同意本协议及竞拍规则中的条款。

3.3 按照竞拍规则缴纳相应的竞拍保证金。

3.4 拍卖人用户，“相信”APP平台未拒绝其发布竞拍品信息申请；竞买人用户，“相信”APP平台未拒绝其参与相关竞拍品竞拍的申请。

第四条 “相信”APP平台的权利和义务

4.1 拍卖人用户和竞买人用户明确知悉：“相信”APP平台在竞拍活动中仅起到竞拍和竞买双方的撮合作用，“相信”APP并不会直接参与用户相关数据资产的竞拍或者竞买，“相信”APP可能会对此撮合交易的收取部分数据资产作为撮合居间费用，亦不会对竞拍或者竞买人交易产生的税费负责。

4.2 “相信”APP平台同意向拍卖人用户和竞买人用户提供稳定的技术服务，使其参与的竞拍得以顺利进行。

4.3 “相信”APP平台有权按照本协议及“相信”APP平台公示规则，在拍卖人用户和竞买人用户支付账户内冻结各类竞拍保证金。

4.4 “相信”APP平台有权按照本协议及“相信”APP平台公示规则，对符合相关条件的拍卖人用户和竞买人用户，采取保证金解除冻结措施。

第五条 拍卖人用户和竞买人用户的权利和义务

5.1 拍卖人用户的权利和义务：

5.1.1 拍卖人用户同意接受“相信”APP平台公示的与竞拍有关的所有规则和流程页面的规定，并同意严格按照前述规定使用“相信”APP竞拍平台技术服务。

5.1.2 拍卖人用户应严格按照本协议及“相信”APP平台相关规则中关于信息发布的要求发布竞拍品。

5.1.3 拍卖人用户应承诺其发布的竞拍品确实为其本人真实所有，且未在其发布的竞拍品上设置任何担保权利，拍卖人用户须对竞拍品的价格等相关信息进行明确说明。

5.1.4 拍卖人用户在竞拍过程中应维护良好的竞拍交易秩序，不得以任何形式操纵或干扰交易。

5.1.5 拍卖人用户应严格按照竞拍结束时的成交价格出售竞拍品。

5.1.6 若竞拍成功后，拍卖人用户具有成交不卖等违规行为，“相信”APP平台有权按照本协议及“相信”APP平台公示规则，对拍卖人用户实施包括但不限于罚没其所缴纳的各类违约金一倍的处罚，用以补偿竞拍成功的竞买人用户。

5.2 竞买人用户的权利和义务：

5.2.1 竞买人用户同意接受“相信”APP平台公示的与竞拍有关的所有规则和流程页面的规定，并同意严格按照前述规定使用“相信”APP竞拍平台技术服务。

5.2.2 竞买人用户应自行妥善保管用户名及密码，该竞拍行为被认为是竞买人用户本人参与竞拍活动的行为，竞买人用户应自行承担竞拍行为产生的相应义务及责任。

5.2.3 参与竞拍活动的竞拍品由拍卖人用户提供，并由拍卖人用户进行竞拍品信息、价格介绍的上传等，以及竞拍成功交付竞拍品流程等。竞买人用户参与竞拍活动前应仔细查看竞拍活动页面对竞拍品的信息、价格描述介绍。

5.2.4 竞买人用户知悉互联网竞拍活动数据处理的时间间隙，如遇系统提示出价金额与最终出价金额不同，竞买人用户认可以最终出价金额为准。

5.2.5 竞买人用户参与竞拍活动成功取得最终购买的竞拍品时，应按竞拍规则完成竞拍商品交易。如未按竞拍规则完成交易的，用户应按竞拍规则承担相应责任及不利后果。

5.2.6 竞买人用户竞拍成功后，应在竞拍规则规定的时间内完成付款。用户竞拍成功后未在竞拍规则规定的时间完成付款，或在竞拍成功并完成货款支付后又申请退款的，“相信”APP平台有权按照本协议及“相信”APP平台公示规则，对竞买人用户实施包括但不限于罚没其所缴纳的各类违约金的处罚，用以补偿竞拍成功的拍卖人用户。

第六条: 竞拍规则

6.1 竞买人用户确认出价金额，点击“出价”按钮进行竞拍。

6.2 竞买人用户可以按照系统提示，在当前价格的基础之上加一个加价幅度出价；也可以手动出价，在当前价格基础上加多个加价幅度出价。用户的出价不能低于当前价格加一个加价幅度的价格。

6.3 出价最高者领先，价低者竞拍记录出局，出局者可以再次出价。

6.4 当一竞买人用户的出价领先时，该竞买人用户不能再次出价或连续加价。直到有更高出价出现后，其方可再次出价。

6.5 对于设置了延时周期的竞拍品，在拍卖时间结束前5分钟内，如有竞买人用户出价，系统将按照页面公示的延时周期时长，自动顺延一个延时周期，给其他竞买人用户考虑和继续出价的时间。如在系统自动顺延期间有竞买人用户出价，系统将再次顺延一个延时周期，直到没有竞买人用户出价为止，拍卖正式结束。

6.6 在拍卖结束后，出价最高者胜出。

6.7 依照国家相关法律规定，拍卖人用户可以对竞拍品设置保留价，且可以不公示保留价的具体金额。有关保留价的法律规定，请参照《中华人民共和国拍卖法》。 拍卖人用户对竞拍品设置了保留价的，胜出者的最终出价如果高于该保留价，则拍卖成交，该胜出者成功竞得该竞拍品；如胜出者的最终出价未达到保留价，则拍卖不成交，该竞拍品流拍，胜出者未成功竞得该竞拍品。如拍卖人用户未设置保留价，系统自动认定成交，胜出者成功竞得该竞拍品。 鉴于“相信”APP竞拍平台所涉及的竞拍品仅为拍卖人用户的个人数字资产，当拍卖人用户对其发布竞拍品设置了保留价，而参与对竞拍品竞拍的竞买人用户最终出价未达到保留价，拍卖不成交时，该竞拍品流拍，流拍后的竞拍品将回到拍卖人用户的账户。
